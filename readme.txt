The LPS_demo.mlx live script presents a practical introduction to LPS.
The work is based on the following papers:
- Jain, Akshay, and James M. Keller. "On the computation of semantically ordered truth values of linguistic protoform summaries." 2015 IEEE International Conference on Fuzzy Systems (FUZZ-IEEE). IEEE, 2015.
- Jain, Akshay, James Keller, and Mihail Popescu. "Explainable AI For Dataset Comparison." 2019 IEEE International Conference on Fuzzy Systems (FUZZ-IEEE). IEEE, 2019.
